# 13.Class

---

传统的Javascript是通过构造函数来定义一个对象；

```
//定义一个构造函数
function Point(x, y) {
	// body...
	this.x = x;
	this.y = y;
}
// 为构造函数添加一些方法
Point.prototype.toString = function() {
	// body...
	return '(' + this.x + ',' + this.y + ')';
};
var p = new Point(1, 2)
console.log(p); //Point { x: 1, y: 2 }
console.log("p is " + p); //p is (1,2)
console.log(typeof Point); //function
console.log(typeof p); //object
```
